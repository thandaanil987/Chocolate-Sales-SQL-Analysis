# Chocolate Sales SQL Analysis

This project contains SQL queries for analyzing a Chocolate Sales database using MySQL.

## Database Setup

```sql
CREATE DATABASE chocolates;
USE chocolates;
```

## Tables Used

* geo
* people_tel
* products
* shipments

## SQL Concepts Covered

### Basic Queries

* View all shipments
* Filter records using WHERE clause
* Sort data using ORDER BY

### Conditional Filtering

* Shipments by specific sales persons
* Shipments to specific regions
* Date range filtering
* IN operator
* LIKE operator with wildcards

### Calculations

* Amount per box calculation
* ROUND function usage

### Joins

* INNER JOIN between shipments and people tables
* INNER JOIN between shipments and geo tables

### Aggregations

* SUM()
* COUNT()
* MAX()
* GROUP BY
* Monthly sales analysis

## Example Business Questions Solved

1. View all shipment records.
2. Find shipments handled by a specific salesperson.
3. Filter shipments by salesperson and geography.
4. Retrieve shipments within a date range.
5. Find products containing specific keywords.
6. Calculate sales amount per box.
7. Analyze shipments for a particular salesperson.
8. Generate monthly sales summaries.
9. Find shipments to a specific country.
10. Determine maximum shipment amount per month.
11. Count shipments by country for a given month.
